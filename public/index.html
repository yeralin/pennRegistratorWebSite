Instructions

Do not worry video tutorial is below!

  1. Register new numerical email on mail.com
        (ex. 8141112345@mail.com)
  2. Change "Text Message Address" on eLion
        in Notification Preferences using this email
  3. Check which radio button (semester)
        is needed for registration
  4. Enter all needed information into the program's
         required fields

Additional instructions about "Drop Option"

  • If you need to drop a class, and register conflict one
  • You simply check "Drop Option" next to "Semester Drop Box"

How does it work?

  • It DOES NOT OVERLOAD PENN STATE SERVERS
  • It simply checks provided email
        for notifications every 10 Seconds
  • It doesn't store provided information ANYWHERE
  • It uses simple, non-visible JS browser
         so you won't actually see how it register classes
